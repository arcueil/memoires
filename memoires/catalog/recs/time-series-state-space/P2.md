## Technique (pymc-labs)

**Built-in PyMC time-series distributions (pm.AR / pm.GaussianRandomWalk / pm.GARCH11)** — PyMC's built-in TS RVs, usable as prior or observed: pm.AR(rho=[...] order-p, constant=True puts intercept as first rho element, init_dist, sigma); pm.GaussianRandomWalk(mu=drift, sigma, steps=T-1, init_dist); pm.GARCH11(omega, alpha_1, beta_1, initial_vol). These hand HMC a differentiable latent chain directly (no manual scan). The page discusses the GEOMETRY of such chains (P1/P2) but never names the API that builds them.
